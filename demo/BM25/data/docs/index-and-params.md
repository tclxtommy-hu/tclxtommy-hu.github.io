# 倒排索引与超参数

稀疏检索的底层依赖倒排索引，而 BM25 的两个超参数 k1、b 直接决定排序行为。

## 倒排索引

倒排索引记录「词项 -> 包含它的文档及词频」的映射（postings），使检索时只需遍历查询词命中的少量链表，复杂度与文档总数解耦。

## k1 参数

k1 控制词频饱和速度，默认 1.5。值越大，词频对得分的影响越「迟钝」，越不容易被重复词堆砌带偏。

## b 参数

b 控制长度归一化强度，默认 0.75。b 越接近 1，长文档被惩罚得越狠；b 为 0 则完全不做长度归一化。

## 调参建议

新闻、问答等短文本可保留默认；长文档语料可适当调高 b 至 0.8~0.9 以抑制长度偏差。
