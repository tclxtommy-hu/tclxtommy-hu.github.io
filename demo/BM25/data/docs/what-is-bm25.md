# BM25 是什么

BM25 是信息检索中衡量「查询词项」与「文档」相关性的经典排序函数，是 TF-IDF 的改进版，广泛用于搜索引擎和稀疏检索。

## 评分公式

对查询 Q 与文档 D，BM25 得分是各词项贡献之和：score(D, Q) = Σ IDF(t) · f(t,D)·(k1+1) / (f(t,D) + k1·(1-b+b·|D|/avgdl))。

其中 f(t,D) 是词项在文档中的词频，|D| 是文档长度（token 数），avgdl 是集合平均文档长度。

## 词频饱和

与 TF-IDF 不同，BM25 的词频项存在饱和：当词频持续增大，相关度贡献趋近于上限 (k1+1)，避免长文档因词频虚高而霸榜。

## 长度归一化

长度归一化项 1-b+b·|D|/avgdl 让长文档的得分被适度压缩，使短而精准的文档更易脱颖而出。
